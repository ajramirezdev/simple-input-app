# Simple Input App

A full-stack app using:

- **Frontend**: React 19, Vite, Tailwind CSS, tRPC
- **Backend**: Express.js, tRPC, Prisma, PostgreSQL

---

### Setup

```bash
# Frontend
cd client
npm install
npm run dev

# Backend
cd server
npm install
npm prisma generate
npm run dev

# server/.env
DATABASE_URL
```
